# Its a GROUP PROJECT
# Women Safety — SafeHer (Full Responsive Project)

## What is included
- Client (HTML + Tailwind CSS + Vanilla JS)
- Server (Node.js + Express)
- MongoDB models for Users and Reports
- Auth with JWT (register/login)
- SOS endpoint (stores lat/lng)
- README and .env.example

## Quick start
1. Unzip.
2. Copy server/.env.example to server/.env and set values.
3. cd server && npm install
4. Start MongoDB (local or Atlas)
5. npm run dev
6. Visit http://localhost:4000
